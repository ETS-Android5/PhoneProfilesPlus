PhoneProfilesPlus Changelog

Version: 3.6.2

  • Added display of activated profile into Editor events view.
  • Added grant permission preference into profile preferences.
  • Added display of "Some underlined parameters are not set" into event preferences.
  • Added display of "No sensor is enabled!" into event preferences.
  • Added display of underlined event parameters with red color, when are not configured.
  • Added display of profile and event name with red color, when required conditions are not set.
  • Added system settings into sensor parameters.
  • Added event status icon for manual profile activation.
  • Added "Never ask again" into alert dialog for check if enable run is checked during save event preferences.
  • Added application parameter "Use alarm clock" for set up profile and event time parameters as exact during screen off.
  • Added indication of sensor fulfillment status in Editor.
  • Added dialog to confirm stop events run.
  • Added "Incoming/Outgoing call end" call event into "Call sensor".
  • Added show of "Quick guide" on first start of application.
  • Removed usage of Device administrator for "Lock device" profile preference.
  • Updated translations.
  • Fixed bugs.
