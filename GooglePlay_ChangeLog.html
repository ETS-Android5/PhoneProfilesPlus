PhoneProfilesPlus Changelog

Version: 3.5.7

  • Added "Wi-Fi hotspot preferences" preference to profile.
  • Added "Pop-up notifications" preference to profile.
  • Added "Background color" parameter for widgets.
  • Added "In call", "Missed call" to "Notification parameters" event sensor.
  • Added enable/disable Wi-Fi, Bluetooth, mobile cells, orientation, location scanning into "Settings".
  • Added profile parameters "Disable mobile cell scanning", "Disable orientation scanning", "Disable location scanning".

  • Changed "Notification sound interval" for start of event norification to seconds.
  • Changed profile preference indicators.
  • Updated translations.
  • Bug fixes.
