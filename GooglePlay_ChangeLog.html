PhoneProfilesPlus Changelog

Version: 3.5.6

  • Removed Accessibility service support due to rule change by Google about using accessibility service. Is replaced with "PhoneProfilesPlusExtender" application.
  • Added "PhoneProfilesPlusExtender" installation info to "Important info".
  • Added root support of Wi-Fi hotspot for Android 8+.
  • Added ability to configure more strings to calendar sensor parameter "Search string".
  • Added notification sound and vibration for default profile activation.
  • Added notification sound and vibration after profile activation duration end.
  • Added usage of profile duration for profile activation from event.
  • Added support for missed calls into event sensor "Call parameters".
  • Added repeating notification for start of event.
  • Added system setting "Keep Wi-Fi on during sleep" into Settings for Wi-Fi scanner.
  • Removed "Duration", "Permanemt Run", "End when dismissed" from event sensor "Notification parameters". "End when dismissed" is used internally.
  • Changed parameter "Cells" in event sensor "Mobile cells parameters". Cell names are grouped.
  • Updated translations.
  • Bug fixes.
