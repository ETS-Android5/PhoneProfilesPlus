PhoneProfilesPlus Changelog

Version: 3.5.5

  • Added "Confirm restart events" to Settings.
  • Added "AutoStart permission" for MIUI ROMs to Settings.
  • Added "Startup delay" to "Run applications" profile preferences.
  • Removed "None" and "Restart events" from Settings about Wi-Fi, bluetooth, mobile cells rescan preference in Settings.
  • Added "Act. without" button to dialog used when profile preference "Ask for duration" is enabled.
  • Changed duration and number pickers in dialogs.
  • Added and changed profile icons.
  • Donation prices are shown in local currency.
  • Changes for Android 8.
  • Decreased battery consumption.
  • Updated translations.
  • Bug fixes.
