PhoneProfilesPlus Changelog

Version: 3.6.7

  • Added "Dark background", "Use decorator", "Layout type" into "Settings"/"Notifications".
  • Battery consumption optimizations, when PPPExtender is used.

  • Moved call receiver for Call event sensor into PPPExtender.
  • Changed display of indicators in profiles and events.
  • Reverted DashClock support.
  • Updated translations.
  • Fixed bugs.
