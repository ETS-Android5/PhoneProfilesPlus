PhoneProfilesPlus Changelog

Version: 3.6.7

  • Added support of night mode for activated profile notification.
  • Added support of night mode for application theme.
  • Added requirement of location enabled for mobile cells scanning in Android 9+.
  • Added "Charger type" preference into event Battery sensor.
  • Added "!" as negative operator into event Calendar sensor for search string.
  • Added intents support into Run applications/shortcut.
  • Added lock device using PhoneProfilesPlusExtender. Supported is Android 9+.
  • Added profile reactivation after profile preferences change.
  • Removed "Wi-Fi hotspot" profile parameter for Android 9. Not working, sorry.
  • Some small GUI changes.
  • Full Android 9 support.
  • Updated translations.
  • Fixed bugs.
