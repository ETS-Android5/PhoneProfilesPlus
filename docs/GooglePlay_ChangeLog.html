PhoneProfilesPlus Changelog

Version: 4.1.3

  • Added button into Editor for configure "Show in Activator" profile parameter.
  • Added "Always-On Display" parameter into profile.
  • Updated translations.
  • Fixed bugs.

Version: 4.1.2

  • Added "Not stopped" events filter into Editor.
  • Added "Sort list" button into "Mobile cells sensor" "Cells" dialog.
  • Added parameter "Vertical position" for "Samsung edge panel" widget.
  • Added new profile icons.
  • Added 2 languages into "Settings"/"Application interface"/"Language".
  • Added send of backup files via e-mail.
  • Added "Support" item into Editor menu.
  • Added "Restart events" icon into small notification layout type.
  • Added notification "Ignore battery optimization" any events are enabled.
  • Changed, in "Mobile cells scanning", parameter "Show notification for detection of not used mobile cell" as enable/disable parameter also for Android >= 8.
  • Allowed profile activation with duration, when "After duration do" is "Activate specific profile".
  • GUI changes.
  • Added support for night mode in Google Maps used in "Location editor".
  • Changes in "Settings"/"Notifications" for Android 10.
  • Updated translations.
  • Fixed bugs.
