PhoneProfilesPlus Changelog

Version: 4.0

  • Removed support for Android 4.4.
  • UI changes.
  • Removed side panel in Editor.
  • Removed shared profile.
  • Added "Alarm Klock" application to Alarm clock Event sensor.
  • Added volume change for DTMF tones, Accessibility prompts, Bluetooth calls into Profile.
  • Language is changed immediately when is selected.
  • Added night mode support for launcher icon.
  • "PhoneProfiles Silent" tone is installed into application internal storage. This not requires Storage permission.
  • Added option to configure tone "None" when profile notification volume is 0.
  • Removed "Color" and "Light with dark title" theme.
  • Updated translations.
  • Fixed bugs.
